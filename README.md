# GodotStateMachine

Godot plugin project for easy state machine creation and development.  

The goal of this plugin is to provide an easy-to-use interface for state machine conception in your games. This interface allows the developper to add states and transition, and generate the corresponding scripts to be implemented by the developper.

Currently, you can 
* Create a State machine that will be added as a child of the currently selected node
* Select a saving directory for the state scripts that belong to this state machine
* Create/rename states that will be added as children of the selected state machine
* Create transition between states
* Delete states or transitions

If you want, you can still break everything, for example by naming different states the same way, but tbh i'm too lazy to fix these kind of details right now and I want to work on another project
